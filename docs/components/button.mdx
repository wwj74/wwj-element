## API

该组件是在 antd 的 [Button](https://4x-ant-design.antgroup.com/components/button-cn/) 基础之上进行封装，antd 原有的 API 全部支持。

### 扩展 API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| primary | 设置主要按钮，和 type="primary" 不同，该方式无背景色 | boolean | - |
| size | 设置按钮大小，只支持传递列出的枚举 | '38' \| '36' \| '34' \| '32' \| '30' \| '28' \| '26' \| '24' \| '22' \| '20' | - |
| type | 扩展了 second-primary、third-primary、warning 类型 | string | - |
| warning | 设置警告按钮，和 type="warning" 不同，该方式无背景色 | boolean | - |
